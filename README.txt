The application uses several different external jars.
All the jars are included in the External jars folder. The images are also all in the Pictures folder.

For the External Jars you will need to add the following : 

httpcomponenets-client-4.5.3/lib/commons-logging-1.2
httpcomponenets-client-4.5.3/lib/httpclient-4.5.3
httpcomponenets-client-4.5.3/lib/httpcore-4.4.6
jfreechart-1.0.19/lib/jcommon-1.0.23
jfreechart-1.0.19/lib/jfreechart-1.0.19
itext-1.4.6
itextpdf-5.4.0
jgoodies-forms-1.8.0
jgoodies-forms-1.8.0-sources
mysql-connector-java-5.1.24-bin

A copy of the database is also in available in the main folder. Import it as a self-contained file (dump).
